# HEARTBEAT RUNNER (Prototype)

**「心拍とシンクロする快感」を検証するためのプロトタイプ**

このゲームは、120BPMの高速な心拍リズムに合わせて障害物を避ける、リズムランナーゲームのプロトタイプです。
「目視不可能な速度で迫る障害物を、リズムを頼りに回避する」というコアメカニクスの面白さを検証することを目的としています。

## ゲーム概要

*   **心拍同期アクション:** 常に一定の心拍（リズム）に合わせてジャンプします。
*   **高速スクロール:** 障害物は一瞬で通り過ぎます。目ではなく、耳とリズム感を信じてください。
*   **ランクシステム:** ミス回数に応じて、あなたの心拍安定度が評価されます。

## プレイ環境

以下の環境でのプレイを推奨しています：

*   **PC** (ブラウザ)
*   **iPad** (横画面 / Landscape)

## 操作方法

| プラットフォーム | アクション | 入力 |
| :--- | :--- | :--- |
| **PC** | ジャンプ | **スペースキー** または **左クリック** |
| **iPad / スマホ** | ジャンプ | **画面タップ** |

## ランク評価

ゲーム終了時、ミス回数に応じて以下のランクが表示されます。

*   **RANK PERFECT:** ミス 0回 (完全なシンクロ)
*   **RANK A:** ミス 1〜2回
*   **RANK B:** ミス 3〜4回
*   **RANK C:** 失敗 (ミス 5回)
