# HEARTBEAT RUNNER (Prototype)

**「リズムとシンクロする快感」を検証するためのプロトタイプ**

このゲームは、リズムに合わせて障害物を避ける、リズムランナーゲームのプロトタイプです。
「目視不可能な速度で迫る障害物を、リズムを頼りに回避する」という体験の面白さを検証することを目的としています。

## ゲーム概要

*   **高速スクロール:** 障害物は一瞬で通り過ぎます。目ではなく、耳とリズム感を信じてください。
*   **ガイドシステム** 緑色のガイドの次の拍でジャンプしてください。

**
## ランク評価

ゲーム終了時、ミス回数に応じて以下のランクが表示されます。

*   **RANK PERFECT:** ミス 0回 (完全なシンクロ)
*   **RANK A:** ミス 1〜2回
*   **RANK B:** ミス 3〜4回
*   **RANK C:** ミス 5回
